写给第200篇博客

发表于 2017-07-29 更新于 2022-10-02 分类于 日志 Valine：

不知不觉已经写了200篇博客了，总要写点东西纪念一下吧。

当然我并不会搞几句华丽的词藻，甚至引用几句诗(eg: 宝剑锋从磨砺出 梅花香自苦寒来)，然后再给自己写几句鸡汤，然后说自己会坚持下去的。这明显不是我的风格唉。

不过话说回来，试问自己，一个多孤独的人，才能坚持写完200篇博客？

当我们在写博客的时候我们在写什么？

写博客是一种表达的方式，类似你拍电影，写小说，画画，制作游戏，拍照，甚至是写代码都是一种表达的方式。我曾经很迷恋“表达方式”这个词，我阅读过一些如何提高漫画表现力的书，漫画的乐趣在于，人们能够通过一张张片段，在脑海中把它们补齐，并播放他们。还读过如何让电影提高表现力的书，而电影就更考验导演的功底了，他不但需要用电影给观众将一个故事，还要让观众完全沉醉在画面，对白与音乐中。同样写作也是一种表达方式。

所以很长一段时间，我都想提高表达方式，比如之前我想找几个学语文学的好的同学，问问他们我的文章写的有什么不足，后来发现，效果不是很明显，后来我也懒的问了，直接去买些教人写作的书去看了。其实我原来想报个教人写作的班的，类似教人画画的班。但是却是没找到好的课程。

后来我觉得这其实也是一件矛盾的事，因为我是写的是技术博客，所以本身我应该更关系，写的是什么，而不是关心用什么方式去表达。所以我认为写好博客，最重要的是，你要让别人理解你像表达的东西，然后才是用什么样的方式去表达。也就是说你的故事是什么，然后才是如何讲出你的故事。

有人说写博客也是一个整理思路的过程。就像你的房间一样，时间长了需要整理。你可能看了一堆东西，这时候就需要用某种手段将它们整理出来。

有人说写博客其实就是把自己看到的书整理出来了而已，并不是什么自己的思想，这一点我非常苟同，但是说实话，我还真不知道什么叫自己的思想，因为我发现我从小到大的所有的思想，都是别人给的。我只不过是一个存放别人思想的容器而已。我的任何思考也都是基于别人的思想。

我还听到不止一个人跟我说这句话：“很多牛逼的人都不写博客的。”。我真的不知道这和我写博客有啥关系，你能从“很多牛逼的人都不写博客的”，推理出“不写博客就能牛逼了”？

还有人会说，你写的东西总有人知道了，没人知道的你又写不出来。这个我还是不知道和我写博客有啥关系。或者说你在学校考不了第一就不考了么？

自我发现

除了自己总结和学习，我认为写博客其实是一个自己反思，自己照镜子的过程。逐步发现自己内心的丑恶。所以本身很痛苦，然而进步就源于痛苦。

这个我在读了黑泽明蛤蟆的油之后，深有感悟。

日本民间流传着这样一个故事：在深山里，有一种特别的蛤蟆，它和同类相比，不仅外表更丑，而且还多长了几条腿。人们抓到它后，将其放在镜前或玻璃箱内，蛤蟆一看到自己丑陋不堪的外表，不禁吓出一身油。这种油，也是民间用来治疗烧伤烫伤的珍贵药材。晚年回首往事，黑泽明自喻是只站在镜前的蛤蟆，发现自己从前的种种不堪，吓出一身油——这油的结晶就是这部《蛤蟆的油》。

写博客看到自己的丑恶其实不仅仅局限于，你发现很多自己以为自己知道的东西，根本写不出来，或者说你发现自己知道的太少。它甚至可能是对你自己之前的认知的否定。比如你以前认为自己做的对的事，其实伤害了许多人。

我记得有一部韩国电影，大概讲的是一个喜欢幻想的女人，认为自己美丽，富有，善良，最后被警方认定为做案凶手，警察说出她有妄想症，所有美好的事物都是她假像出来的，然后拿出镜子让她仔细看，当她看到镜子里丑恶，臃肿的脸时，吓哭了。

其实你去发博客，去知乎刷帖，与别人聊天，甚至是写发朋友圈，或者你什么都不表达，仅仅是看别人表达，你都能从点点滴滴的细节中发现自己的不足。

比如在知乎上，你能看到很多强答的博主，什么都要回答，显得什么都知道，但其实都暴露了自己的无知。还有些人，只言片语都暴露了自己内心的丑恶，道德的败坏。

但就是这样，你还是要坚持去写博客，去发朋友圈，去跟人聊天，不断的逼出自己的油，被自己的丑陋吓到，不断的痛苦，因为这样你才能成长，否则一辈子蒙在鼓里。

谁会看我的博客

其实吧，我真没指望有多少人看我的博客，毕竟如果我想让别人看，我就去百度做seo了，而不是在朋友圈发链接了，虽然很长一段时间我热衷发朋友圈，但是我从来不会点开别人朋友圈的文章，别说文章懒得点，就是照片我都懒得点，所以我认为，我既然这么对待朋友圈，其他人也一定这么对待。事实就是这样，朋友圈里有人发自己参加了别人婚礼，有人说自己在地铁遇到了傻逼，还有人转载一些金融和房价的文章，甚至还带上自己的评论，还有人说自己生病了很难受，人与人的悲欢离合并不相通，谁会在意呢。所以我发朋友圈的最大的目的，其实是逼迫自己更新博客，当然我其实还关心自己从写这篇博客中可以收获到什么，然后才是其他人能得到什么，毕竟我是自己搭建的博客系统，自己花钱买的域名，没必要讨好任何人，让他们来点击我的广告。

为什么喜欢写博客

有一种对游戏玩家的分类方法叫：巴特尔游戏玩家分类。虽然这个分类在今天看来确实不严谨，但是确实能说明很多问题。这四个分类是，成就型，探险型，社交型，杀手型。

对探险型玩家（Explorers）的描述是，尝试在游戏中寻找一切他们所能找到的东西，据说很多游戏设计师都属于这一类型，所以我觉得喜欢写博客的人，也是这类玩家。

我发现我玩游戏的时候，也对剧情和游戏世界沉迷，想要不断的探索，然后告诉别人，所以我写博客肯定是类似的动机，就是想去探索发现一些东西，然后告诉分享给别人。这是我内心的喜好。

还能坚持多久？

我大学关注过几个牛逼的人的博客，貌似他们自从结婚生子之后，就不怎么更新了。我还关注过几个在b站上做电影和游戏视频的博主，每次都会讲很多干货，但是他们有一部分坚持到了20多集就没有再次更新了。
其实我觉得一开始，每个人都有表达的欲望，可惜随着新鲜感的降低，很多人慢慢也就坚持不下来了。所以有些导演也是拍了一步电影，就撤了。用他们的话说就是过把瘾。
很多人说兴趣是最大的老师，可惜这个老师带你入门之后就走了，到后来，唯有克己才是。

下一步怎么办？

“人类天生自带厌恶的天赋，这逼着他们去寻找新事物。”

你是否有这种感觉，虽然很多人天天的在朋友圈转发各种公众号，可你就是不想点进去，因为你看到题目就知道它想要说什么。

如果你关注过一些流量高的公众号，你会发现他们很多人，把一个观点能写成十篇文章，或者说，他们说来说去，就是那么一套道理。

这被我称之为死循环，我跟许多人说过，我最怕的就是死循环。许多人看书的时候，也是死循环，不是有句话叫：“很多人只想看到他们想看到的。”，所以有一个观点是，你读了再多的书，都是在读你想读到的。然后试图去强化自己的观点罢了。

也许是因为那些公众号为了提高流量不得不这样写吧。我真不想成为这样的死循环，或者说我经常问自己，是不是已经陷入这种死循环了。

所以我希望自己进步，而不是原地止步不前。写了200篇博客，总归是要尝试更多的改变的，而不是目标是写300篇博客。

如果你看了我的博客，会发现我最近的三篇博客都是和编程无关的。一部影视，一部游戏，一部关于思考编段子的。因为我希望以后再也不写类似的文章了，这次就算是收尾谢幕。

打算转型的原因是，我可不想成为那种看似什么都懂，但是什么都干不成的loser。所以我最近要只专心与技术编程，不扯那么些淡了，多才多艺是好，但是不是谁都玩得起的。

所以说，到底有什么打算吧，我就是觉得现在挺累的，想先好好睡一觉。

然后等我起来之后，之前的两百篇博客写的好还是不好，我都要开始新的规划。

比如说只专注于写技术博客，不写游戏和电影之类的。

记得很早以前，我一直想成为一个有趣的人，我不知道写博客到底能不能让我成为一个有趣的人，但最起码现在我并不有趣。
