为啥太刀这么长

发表于 2017-06-11   |   分类于 游戏知识   |  

如果你看过一些日本的武士片或者是玩过一些类似《怪物猎人》，《仁王》，《荣耀战魂》这样包含日本元素的游戏，那你一定见过那种长长的，被称为太刀的武士刀。这篇博客就来考据一下为啥这种刀这么长。

首先我查阅了一些资料发现普通的太刀并没有那么长，也就是 2尺（66cm）以上、3尺（约1m）之间。可以说差不多一米左右。所以这种刀经常会和另一种长度相近的打刀搞混。

太刀与打刀

先说太刀，太刀出现的时代要比打刀早，出现在日本的平安时期（相当与从中国中唐到南宋中前期），当时的战争形式主要是马战。你在日本的游戏或者电影中一定见过类似穿着盔甲骑着马，腰间挎着刀的战士。而这种就是太刀。

而打刀出现在，室町时代到江户时代（15世纪后半刀19世纪中叶），那时主要是地面战斗。所以打刀主要是给站立在地上的武士使用的。

区分太刀和打刀的主要方式是佩挂和刀刃的方向。

首先太刀的刀刃是冲下的，据说一来是符合骑马时候的把刀姿势，（拔出刀后，刚好刀刃向前），二来是为了防止刀刃冲下会刺伤马。所以太刀的刀装是有吊环的。

而打刀的佩戴则是刀刃冲上。这样在把刀的时候，不用像太刀那样需要先举的很高才能砍下来，而是直接砍出来，或者说挥出来更合适。所以打刀的刀装是没有吊环的。

之所以这样，是因为马战一般都是提前双方做好准备然后开始两军交战的。而打刀更有一种一言不合就拔刀的感觉，所以它需要拔刀更快。

除了看刀刃方向，我还查阅到一个小细节，就是太刀因为是要砍对面穿盔甲的人，所以刀刃没有打刀那么锋利（但是也很锋利），否则砍到盔甲上刀刃就坏了。

这么说来，电影游戏上，那些穿着盔甲，配着刀的武士，他们腰间的都是太刀。而我们看的很多电影上，穿着和服佩戴武士刀的武士，他们佩戴的都是打刀（比如《椿三十郎》和《用心棒》）。

现代电影中，也会出现拿武士刀的桥段，比如《杀死比尔》，而这也算是打刀。

居合

说道”一言不合就拔刀“，很多武士片都有这样一个场景，就是两个人面对面，准备决斗，双方摆好架势，然后速度快的一方一招解决掉对手（这种方式被称为居合）。这个桥段后来被美国电影借鉴过去拍西部片了，就是两个牛仔，看谁拔枪快，先解决掉对方。

前面提到了居合，虽然说打刀的出刀速度快，但是还有一种更快的打刀出刀方式。如果你看过黑泽明的《椿三十郎》就会知道。

在片中三船敏郎扮演的三十郎和仲代达矢扮演的 室户半兵卫最后决斗。
可以看到双方都是刀配左边，室户半兵卫 采用标准的打刀出刀方式，右手先拔刀再挥刀。而三十郎则是左手把刀后，直接用右手手臂把刀推了出去。

当然这里也暗指室户半兵卫是标准的武士，从佩刀就可以看出，他配了两把刀，还有一把短的肋差。
而三十郎是个浪人，游侠，他可以选择不按套路出牌。

大太刀

但是我记得在游戏中和影视中确实见过一种很长的刀，比如在《七武士》中，三船敏郎扮演的菊千代的那把显眼的武士刀。

这种太刀被称为野太刀，或者是大太刀，有三尺多刀五尺的。首先能挥舞这种刀的人，一定身体素质要非常好（臂力和腕力足够大）。技巧非常高，才能使用的好这种刀。而且这种到砍下去的威力自然也会很大。

这种刀的刀柄会很长，方便双手挥砍，并且都是背在背后的，跨在腰间太不方便了。

所以《荣耀战魂》的武士和《怪物猎人》里面佩戴的太刀，都算是野太刀。

刚才说道了《七武士》中菊千代。别的武士都佩戴打刀，唯独他拿着一把大太刀，并且蹩脚的像扛锄头一样的扛着刀。其实是导演黑泽明暗示观众，他其实是农民出身，而不是武士出身。
