markdown语法

标题

# 一级标题

## 二级标题

### 三级标题

#### 三级标题

##### 四级标题

###### 五级标题

####### 六级标题

注: 在#和[一级标题]之间建议保留一个字符的空格，这是最标准的Markdown写法。

效果:

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

列表

普通列表

- 文本1

- 文本2

- 文本3

效果:

  • 文本1
  • 文本2
  • 文本3

有序列表

\1. 文本1
\2. 文本2
\3. 文本3

效果:

  1. 文本1
  2. 文本2
  3. 文本3

注: -，1. 和文本之间要保留一个字符的空格。

链接和图片

[显示文本](链接地址)

[豆瓣](http:\www.douban.com)

![](图片链接地址)

![](http:\xxx.xxx.xxx\xxx.jpg)

效果:

豆瓣

注: 普通链接和图片链接很像，只是多了一个!。

粗体和斜体

*斜体*

**粗体**

效果:

斜体
粗体

注：三个以上的***就是华丽的分割线。

程序代码

单个的字符(单词)转义

`python`

效果:

python

成段的代码使用Tab键，或者是四个空格。

四个空格 print “hello,world!”
四个空格 print “i love you!”
四个空格 print “i think so!”

效果:

print "hello,world!"
print "i love you!"
print "i think so!"
